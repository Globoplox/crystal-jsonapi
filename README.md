# json_api

[JSON API](http://jsonapi.org) Serializers for Crystal

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  json_api:
    github: jreinert/crystal-jsonapi
```


## Usage

See [examples/people.cr](examples/people.cr) for a real-world example.
